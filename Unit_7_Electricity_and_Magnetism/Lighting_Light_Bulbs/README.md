# Lighting a Bulb Lab

This interactive lab allows students to explore the fundamentals of electrical circuits by connecting a battery to one or more light bulbs using wires.

## Features
- **Interactive Canvas**: Drag and drop batteries and bulbs.
- **Wire Drawing**: Connect terminals to form circuits.
- **Circuit Simulation**: Real-time evaluation of circuit connectivity and bulb brightness.
- **Dynamic UI**: Visual feedback for lit bulbs and short circuits.

## How to Use
1. **Add Bulbs**: Use the "Add Bulb" button to place more bulbs on the canvas.
2. **Connect**: Click and drag from one terminal (blue circle) to another to draw a wire.
3. **Interact**: Drag components to move them. Select a bulb to see rotation and deletion controls.
4. **Navigation**: Drag the background to pan and use the scroll wheel to zoom.
