The Big Picture: Is this a computer?

It's hard to see how turning on a lightbulb translates to a modern machine. Modern computers just combine billions of these microscopic gates to do three main things:

  • 1. Math (The Brain) Look at the Mini Calculator challenge. By combining 7 logic gates, we created a circuit that can literally add numbers! Chain 32 of these block-groups together, and you have a processor that adds millions in a fraction of a second. This is the core of a CPU.
  • 2. Memory (The Notebook) Look at the SR Latch challenge. Notice how the lightbulb stays ON even after you turn the switch OFF? The gates trapped the electricity in a loop. By doing this billions of times, we create computer RAM.
  • 3. Data (The Alphabet) Look at the Binary Numbers challenge. If we group 4 wires together, a specific pattern of ON/OFFs can represent numbers. If we group 8 wires (a Byte), it can represent a letter like 'A', or a single red pixel on your screen!

The Basics

Computers run on electricity. Wires can either have power (ON / 1) or no power (OFF / 0). Logic gates look at the state of their input wires and use simple mathematical rules to decide whether their output wire should be ON or OFF.

AND Gate

The output is ON only if BOTH inputs are ON.

Input A Input B Output
0 0 0
1 0 0
0 1 0
1 1 1

OR Gate

The output is ON if AT LEAST ONE input is ON.

Input A Input B Output
0 0 0
1 0 1
0 1 1
1 1 1

NOT Gate (Inverter)

Reverses the signal. If input is ON, output is OFF.

Input A Output
0 1
1 0

XOR Gate (Exclusive OR)

The output is ON if EXACTLY ONE input is ON.

0 0 0
1 0 1
0 1 1
